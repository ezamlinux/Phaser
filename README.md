# ENDLESS RUNNER

## Made with Phaser.io
Simple game made with Phaser.io and ES6 Boilerplate

## Screenshot
![Screenshot](screen.png)

## Installation
`yarn && yarn serve`


# Rules
Blue potion : score multiplier

Green potion : retrieve 1 HP

Red potion : retrieve all HP

Yellow potion : + 1 HP Max
